🚀 DÉMARRAGE RAPIDE

EN DÉVELOPPEMENT (actuellement actif) :

   Backend:  cd backend && npm run dev
   Frontend: cd frontend && npm run dev

   Accès: http://localhost:3000

EN PRODUCTION (avec Docker) :

   1. Générer secrets:     ./scripts/generate-secrets.sh
   2. Configurer .env:     nano .env.docker
   3. Déployer:           ./scripts/deploy-production.sh
   4. Vérifier:           ./scripts/health-check.sh

🔐 SÉCURITÉ - POINTS CRITIQUES

AVANT DE DÉPLOYER EN PRODUCTION :

  ⚠️  Générer des secrets UNIQUES :
      ./scripts/generate-secrets.sh

  ⚠️  Configurer HTTPS (Let's Encrypt) :
      sudo certbot certonly --standalone -d votre-domaine.com

  ⚠️  Configurer le firewall :
      sudo ufw enable
      sudo ufw allow 22,80,443/tcp

  ⚠️  Configurer backup automatique :
      crontab -e
      0 2 * * * /path/to/scripts/backup-mongodb.sh
